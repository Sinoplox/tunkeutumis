## B)
Kattelin vähän netistä ja tekoälyltä ja mukama syntaxi seuraa ISO/IEC 7816-4 standardia.

En kyllä ihan ymmärtäny mihin tätä tietoo minun pitää soveltaa:

CLA  INS  P1  P2  Lc  Data  LE

| Field   | Size   | Meaning                                                                |
| ------- | ------ | ---------------------------------------------------------------------- |
| CLA | 1 byte | Class of instruction (defines logical channel, secure messaging, etc.) |
| INS | 1 byte | Instruction code (what operation to perform)                           |
| P1  | 1 byte | Parameter 1 (instruction-specific)                                     |
| P2  | 1 byte | Parameter 2 (instruction-specific)                                     |
